Introduction to JavaScript

JavaScript is a high-level, dynamic, untyped, and interpreted programming language. It is a core technology of the World Wide Web, alongside HTML and CSS, and enables interactive web pages. JavaScript is essential for creating web applications and can be used on both the client-side and server-side. Its versatility allows it to be used for various tasks, including web development, game development, mobile app development, and more.

Introduction to Functions in JavaScript

A function in JavaScript is a block of code designed to perform a particular task. It is executed when "called" (invoked). Functions can take parameters and return values.

function add(a, b) {
    return a + b;
}
console.log(add(2, 3)); // Output: 5

Introduction to Methods, Loops, and Switches in JavaScript

Methods

Methods are functions that are associated with an object. They can be invoked using the dot notation.

let person = {
    firstName: "John",
    lastName: "Doe",
    fullName: function() {
        return this.firstName + " " + this.lastName;
    }
};
console.log(person.fullName()); // Output: John Doe

Loops

Loops are used to execute a block of code repeatedly. Common loop types include for, while, and do...while.

for (let i = 0; i < 5; i++) {
    console.log(i); // Output: 0 1 2 3 4
}

Switches

The switch statement is used to perform different actions based on different conditions.

let day = 2;
switch (day) {
    case 1:
        console.log("Monday");
        break;
    case 2:
        console.log("Tuesday");
        break;
    default:
        console.log("Another day");
}

Introduction to Objects in JavaScript

Objects are collections of properties, where each property is defined as a key-value pair. Objects can represent real-world entities.

let car = {
    make: "Toyota",
    model: "Camry",
    year: 2021,
    drive: function() {
        console.log("Driving...");
    }
};
car.drive(); // Output: Driving...

Introduction to Array and Call Stack

Arrays

An array is a data structure that can hold multiple values under a single variable name. Arrays can store elements of different types.

let fruits = ["apple", "banana", "cherry"];
console.log(fruits[1]); // Output: banana

Call Stack

The call stack is a mechanism for an interpreter (like JavaScript's) to keep track of function calls. It operates in a last-in, first-out (LIFO) manner.

function first() {
    second();
}
function second() {
    console.log("Hello from second!");
}
first(); // Output: Hello from second!

Introduction to HTML DOM

The HTML Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, allowing programming languages like JavaScript to manipulate the content and structure of web pages.

document.getElementById("myElement").innerHTML = "Hello World!";
