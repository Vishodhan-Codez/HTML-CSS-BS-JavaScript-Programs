Control Structures

Conditionals

Control the flow of your program with if statements and switch.

if (age > 18) {
    console.log('Adult');
} else {
    console.log('Minor');
}

Loops

Loops allow you to run a block of code multiple times.

for (let i = 0; i < 5; i++) {
    console.log(i); // Outputs numbers 0 to 4
}
