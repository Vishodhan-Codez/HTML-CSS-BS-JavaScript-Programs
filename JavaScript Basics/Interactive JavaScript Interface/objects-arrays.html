Objects and Arrays

Objects

Objects store data as key-value pairs:

const person = {
    name: 'Alice',
    age: 30
}; // 'name' and 'age' are properties

Arrays

Arrays hold ordered lists of values:

const fruits = ['Apple', 'Banana', 'Cherry']; // A list of fruits
